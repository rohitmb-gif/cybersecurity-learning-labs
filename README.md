# Cybersecurity Learning Labs

This repository documents my structured journey into cybersecurity from a non-IT background.

The focus is on:
- Core fundamentals (networking, operating systems, web)
- Hands-on labs and experiments
- Clear explanations and reflections
- Ethical and legal learning practices

## Learning Areas

- Networking (Packet Tracer, Wireshark)
- Linux command line (Kali Linux)
- Web fundamentals (HTTP, requests, responses)
- TryHackMe learning paths
- Basic scripting (Python, Bash)

## Structure

- `networking/` – Networking concepts and labs
- `linux/` – Command-line practice and notes
- `web/` – Web fundamentals and tooling notes
- `tryhackme/` – Learning path reflections and lab summaries
- `scripting/` – Small automation scripts
- `reflections/` – Weekly learning summaries

## Disclaimer

This repository does not contain:
- Exploit payloads
- Flags or answers from labs
- Any illegal or unethical material

All content is for educational purposes only.
